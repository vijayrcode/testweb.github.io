Redefining Team Fitness

Warm Ups is an innovative, interactive fitness game that transforms ordinary workouts into thrilling team-building challenges. Unlike conventional systems requiring external sensors, our solution leverages Android devices as glowing touchpoints, connecting multiple devices via 4G/5G networks.

This cutting-edge approach enables:

  • Seamless participation for individuals, athletes, and corporate teams
  • Minimal setup requirements with all engagement
  • Customizable high-intensity fitness activities

Game Variations

Warm Ups features progressive difficulty variations to maintain engagement and physical challenge. Each variation includes clear benefits and implementation guidelines.

Beginner Variations

Tiptoe Walk Variation

Movement: Walk/run on tiptoes to target

Benefits: Improves balance and calf engagement

One-Legged Hop Variation

Movement: Unilateral hopping with leg switch after touch

Benefits: Enhances coordination and lower-body strength

Intermediate Variations

Crouch Walk Variation (Duck Walk)

Movement: Squat-position locomotion

Benefits:

  • Strengthens quadriceps, glutes, hamstrings
  • Improves stability through low center of gravity
  • Enhances hip/ankle mobility

Advanced Variations

Wheelbarrow Walk Variation

Movement: Partner-assisted hand walking

Benefits:

  • Builds upper-body strength
  • Fosters teamwork and trust

Implementation Guide

Game Workflow: Seamless Setup Process

  1. Installation
    • Download from Google Play Store on all devices
  2. Initialization
    • Launch and select Single Glow/Two Glow mode
  3. Game Configuration
    • Choose Count-Based or Duration-Based format
  4. Network Setup
    1. Generate unique game code on host device
    2. Join session on participant devices
  5. Device Placement
    • Arrange according to selected variation
  6. Game Execution
    • Initiate and monitor session
